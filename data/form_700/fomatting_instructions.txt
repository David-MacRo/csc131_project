The matcher reads from relevant columns in the .xlsx file to aquire interest names to match against.
However, standardizing the format of these inputs is a largely intractible task.

The matcher takes the input interest names from:
- A1 column NAME OF BUSINESS ENTITY
- A2 column NAME OF BUSINESS ENTITY OR TRUST
- C column NAME OF SOURCE
then converts the names to lowercase, and splits them on the # symbol

For example:
"HPQ#Hewlett-Packard#Hewlett Packard"

will match with:
"hpq", "hewlett-packard", or "hewlett packard"

Reccomendations:
- include names, but remove "llc" "co" or similar
- offer different versions for hyphenated names, as spaces are commonly used as an alternative
- include stock tickers if you care about them, but you probably don't tbh
- don't include abbreviations which are short enough to commonly occur in other words: ex "hp"
